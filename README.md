# HR-Data-Analysis-Excel-PROJECT
Interactive HR Analytics Dashboard developed in Microsoft Excel to analyze employee demographics, performance, engagement, attrition, recruitment effectiveness, and workforce trends using Pivot Tables, Pivot Charts, Slicers, and advanced Excel analytics.

👥 Human Resources Analytics Dashboard
📌 Project Overview
This project focuses on analyzing workforce data to uncover insights related to employee performance, engagement, recruitment effectiveness, retention, and organizational productivity.
The dashboard provides HR managers and decision-makers with actionable insights to improve employee satisfaction, reduce attrition, and optimize workforce management strategies.

🎯 Business Problem
The HR department aims to answer key questions regarding:

Employee distribution across departments
Workforce demographics
Employee performance trends
Attrition and termination patterns
Recruitment source effectiveness
Employee engagement and satisfaction
Attendance and punctuality analysis

The objective is to support strategic HR decision-making through data-driven insights.
